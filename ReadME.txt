I use webpack (NodeJS) to deploy app, you can change port by open file webpack.config.js change uncomment line and set your port.

Data config file website in folder data.

Component files in folder src/component.

Code was written by Reactjs and ES6 class.

Css, js, fonts, images files in folder public.

To start web: open cmd to folder, run "npm install", then run "npm start", open "http://localhost:8080/"
